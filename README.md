# NewManagement
jsp期末设计 新闻管理系统

采用前后端分离的写法

前端使用vue编写 后端使用jsp servlet + mysql

由于只是用于混一下作业，只满足了要求实现了基本的数据的增删改查，用户什么的可以去掉的都去掉了
